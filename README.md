-BananaPay

Sistema bancário web desenvolvido com HTML, CSS, JavaScript ultilizando o Supabase como hospedagem do banco de dados.

-Funcionalidades
- Cadastro de usuários
- Login
- Depósito
- Saque
- Transferência
- Histórico de transações

-Tecnologias
- HTML/CSS/JavaScript
- Supabase (banco de dados)

-Como usar
1. Clone o repositório
2. Abra o `index.html`  no navegador
3. cadastre e ultilize das funcionalidades
